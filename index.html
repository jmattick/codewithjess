About Me

I am a recent college graduate developing coding skills in my free time. Currently, I am focusing on improving my JavaScript skills. I try to learn something new from each project I complete and make sure that I can communicate clearly what my code means. I apply the old latin principle Docendo discimus ("by teaching, we learn") to my studies. As I learn new skills, I solidify my understanding by describing what I've learned in how-to youtube videos.

Portfolio

RGB Color Game

This project tests a user's rgb color knowledge by displaying tiles of varying colors and an rgb color code. The user must click on the correct colored tile. If the user is incorrect, the tile will disappear. If the user is correct, the title background will match the correct color and a new round will begin.

Simon Game

The final project in Free Code Camp's Advanced Front-End Development projects leading to the Front-End Development Certificate. This app mimics the classic Simon game by requiring users to match a sequence of buttons to what they observe from the computer. An important aspect of developing this app was to coordinate the timing sequences.

Pomodoro Clock

This project was part of Free Code Camp's Advanced Front-End Development projects. This clock allow a user to work in a Pomodoro style by letting the user set a timer for work time and a timer for rest time. I used HTML5 canvas to create a clock animation that moves proportionally to how much time is left on the timer.

JavaScript Calculator

This project was part of Free Code Camp's Advanced Front-End Development projects. I wanted to emulate a functional calculator on a desk using JavaScript and CSS. I personalized it by creating a cup of coffee and adding a notepad that a user can take notes on. As well as completing basic arithmetic, a user can click on the i button to get more info about the calculator.

Wikipedia Viewer

I created this Wikipedia Viewer as part of Free Code Camp's Intermediate Front-End Development challenges. A user can search for any term and this viewer will show links to relevant wikipedia entries as well as previews to the articles. This project solidified my understanding of using JSONP.

Random Quote Machine

I created this Random Quote Machine as part of Free Code Camp's Intermediate Front-End Development challenges. A user can press a button to recieve a random quote and then press another button to tweet that quote. This project was my first time working with an API.
